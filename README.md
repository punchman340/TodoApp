# Todo Web Application
A Task Management Web application built with Spring boot, PostgreSQL, and Vanilla JS


# Project Overview
This is RESTful API-base Todo Management system.
This project is designed with a 3-Layer Architecture.

# USE TECH
Backend
* JAVA 21
* Spring boot 4.0.2
  - Spring Web(REST API)
  - Spring Data JPA(ORM)
  - Spring Validation(Server side input validation)
* POSTGRESQL 17
* LOMBOK
* Gradle 8

Frontend
* HTML5/CSS3
* Vanilla JavaScript

# Main features
* CRUD Operations(Create, Read, Update, Delete)
* Toggle status(Completed or NOT)
* Task Filtering(Completion status)
* Keyword Search(by title or description)
* Responsive Web Design(Optimized for screen size)
* RESTful Architecture(Maybe)
